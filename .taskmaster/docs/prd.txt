# Product Requirements Document (PRD)

## 프로젝트명
자동 블로그 포스트 생성기 (Auto Blog Post Generator)

## 개요
- Python 스크립트를 사용하여 지정된 주제, 키워드, 또는 외부 데이터 소스를 기반으로 Jekyll 블로그용 마크다운 포스트를 자동으로 생성한다.
- 생성된 포스트는 `_posts/` 디렉터리에 저장된다.

## 주요 기능
1. 주제/키워드 기반 포스트 자동 생성
2. 외부 데이터(예: RSS, API, CSV 등)로부터 포스트 생성
3. 포스트 메타데이터(제목, 날짜, 태그 등) 자동 작성
4. 포스트 파일명 및 포맷 Jekyll 규칙 준수
5. 스케줄링 또는 수동 실행 지원

## 사용자 시나리오
- 사용자는 키워드 또는 데이터 소스를 입력한다.
- 스크립트가 자동으로 포스트를 생성하고 `_posts/`에 저장한다.
- 사용자는 생성된 포스트를 검토 후 배포한다.

## 비고
- Python 3.x 환경
- Jekyll 블로그 구조와 호환
- 확장성 고려(다양한 데이터 소스 지원)
